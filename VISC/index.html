Valentin's I3 Shell Script made for beginners and for people that don't want to spend the rest of eternity configuring their system.

Supported distros are: Arch Linux and Debian and Debian based distros. You can still use VISC but you'll have to install the packages manually, which is really easy.

Support for other distributions will come soon enough.

Installation tutorial

First you need to clone the Github repository, if its not pre-installed you need to install git for this step. To clone the repository:

git clone "https://github.com/vdozer/VISC"

Next you need to change directories

cd VISC

To run it:

sh visc.sh

There aren't many steps so you'll be okay if you're a beginner. At the end it will ask you if you want to use zsh or not.

Keybinds

mod means Windows key.

  • mod+Return - Start terminal (Alacritty)
  • mod+t - Start file manager (Thunar)
  • mod+Shift+p - Take screenshot (Flameshot
  • mod+Shift+w - Start browser (firefox)
  • mod+Shift+q - Kill focused window
  • mod+Shift+h - Move focused window left
  • mod+Shift+j - Move focused window down
  • mod+Shift+k - Move focused window up
  • mod+Shift+l - Move focused window right
  • mod+h - Focus left
  • mod+j - Focus down
  • mod+k - Focus up
  • mod+l - Focus right

Alternatively, you could use the arrow keys.
